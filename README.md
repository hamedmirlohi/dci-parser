1. Clone this repository:

    ```
    git clone git@github.com:hamedmirlohi/dci-parser.git
    ```

2. Change directory to new project.
3. Install project dependencies

    ```
    bundle install
    ```

4. Prepare local database

    ```
    rake db:create
    rake db:migrate
    ```

5. Run server

    ```
    rails server
    ```
